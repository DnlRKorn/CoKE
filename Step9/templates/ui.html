Abstract: Coronavirus Knowledge Extractor (CoKE) web application tool provides the scientific community with (i) data on human and viral proteins associated with SARS-CoV-2 and other coronaviruses that are indexed in UniProt and (ii) chemicals targeting those proteins, which could potentially assist with drug repurposing for COVID-19. The CoKE tool is built off the COVID Open Research Dataset (CORD-19), SciBiteAI's ontological tagging software, UniProt, and DrugBank. A custom algorithm identifies co-occurrences of protein and drug terms within coronavirus-related papers in CORD-19. The score provided to the user rates the confidence of a co-occurrence. The closer the score to 0, the higher the confidence. Informed utilization of literature by means of curation and ontological tagging allows dynamic tools which can provide sentence-level data to be created in a short amount of time. The CoKE tool and dataset enables rapid searching and exploration of drug-to-protein connections by curating large datasets of full text papers, filtering, and providing a dynamic web portal to users. The code is available at https://github.com/DnlRKorn/CoKE.
